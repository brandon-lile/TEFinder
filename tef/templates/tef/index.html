Sequence Search

How to Use

Use the TEFinder to find transposons in your DNA sequence. Simply enter your DNA sequence in question below and hit enter.

  • You can customize your search by specifying a start and ending region to search through.
    • 0 represents the first base in the sequence.
    • 0 for start location will start from the beginning.
    • 0 for end location will search to the end of the sequence.
  • Threshold allows you to specify the minimum number of matches before declaring it a successful match.
    • 3 is the recommended minimum
